What's Codymer?

This a Dart SASS architecture for bringing light to your Front's Design Systems.

Download V.4.2.1 Documentation

No JS depencies

Codymer has no Javascript dependencies. This is why there is not accordions, tooltips, modals or other dynamic elements. Feel free to add what you need.

BEM Methodology

The classes system is based in BEM methodology, for a better and solid classes, SASS and styles properties management.

Mobile First

Codymer's coding is Mobile First oriented, so min-width over max-width media-queries.

CSS Lint, W3C and Project Wallace

For a better and optimized CSS file Codymer tries to follow the rules and recommendations of:

  • csslint.net
  • jigsaw.w3.org/css-validator
  • projectwallace.com
