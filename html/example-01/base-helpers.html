The Helpers

These are classes you can use repeatedly in all your themes. Some of them work the same way, no matter the theme, and others work different depending on the properties you set in the setup file.

Size

This classes can be set in the setup file to create custom width and height properties. This properties can change depending of 7 different breakpoints you add in the @include helper-size();.

Here it's an example im the styles-theme-codymer.css file.

  • .helper-size-100

Animation

  • .helper-animation-fadeIn

Gap

Like in Size helper, Gap depends on what you set in the setup file of each theme. You can create custom spaces based in css margin property, at the same time this classes can be set as 7 different breakpoints media-queries. This class can be set with @include helper-spacer-gap();.

Here it's an example im the styles-theme-codymer.css file.

  • .helper-spacer-gap-t-100 This is an example spacer gap top density of 100

Gap inner

This class works the same way as gap but using css padding.

Here it's an example im the styles-theme-codymer.css file.

  • .helper-spacer-gap-t-100 This is a spacer gap top density of 100

Background

They are divided in 2 categories. The ones work the same way no matter the theme:

  • .helper-bg-color-black
  • .helper-bg-color-white
  • .helper-bg-color-blue-facebook-500
  • .helper-bg-color-blue-instagram-500
  • .helper-bg-color-blue-twitter-500
  • .helper-bg-color-green-whatsapp-400
  • .helper-bg-color-green-whatsapp-500
  • .helper-bg-color-purple-instagram-500
  • .helper-bg-color-transparent-(mo, sm, md, lg)

And the other group that can change depending on the theme. You can set them in the setup file with the @include helper-backgrounds();:

  • .helper-bg-color-primary
  • .helper-bg-color-secondary
  • .helper-bg-color-tertiary
  • .helper-bg-color-quaternary
  • .helper-bg-color-quinary
  • .helper-bg-color-senary
  • .helper-bg-color-septenary
  • .helper-bg-color-octonary
  • .helper-bg-color-nonary
  • .helper-bg-color-denary
  • .helper-bg-color-undenary
  • .helper-bg-color-duodenary
  • .helper-bg-color-trendary

Additionally you have to control the css background-size properties:

  • .helper-bg-auto
  • .helper-bg-cover
  • .helper-bg-contain

With the @include helper-backgrounds-color-behavior(); you can control the behavior of the elements inside a background helper:

  • .helper-bg-color-black *
  • .helper-bg-color-senary *
